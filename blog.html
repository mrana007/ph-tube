Discuss the scope of var, let, and const

var, let and const all are variable. var declarations are globally scoped. var variable can be updated and we can re-declared var variable. on the other hand let and const are blocked scope. let can be updated but we can not re-declared it. const variable can not updated also we can not re-declared const variable.

Tell us the use cases of null and undefined

in null case variable has empty value on the other hand in the case of undefined means variable is declared but the value is not assigned for the variable
